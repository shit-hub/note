          官方


          需要先安装某一版本的 Go，且切换不是很方便

          前提条件：

          • git
          • go

          安装 #

          使用具体版本号代替 X.Y.Z

          go install golang.org/dl/goX.Y.Z@latest
goX.Y.Z download

          使用 #

          goX.Y.Z version

          如需替换默认的 go 命令，建议使用软连接去链接新版本的 go 可执行文件

          卸载 #

          goX.Y.Z env GOROOT

          删除得到的 GOROOT 路径以及 goX.Y.Z 的二进制文件即可

          参考链接： https://go.dev/doc/manage-install
